# 🥊 CSA Practice Hub — 300 Questions

> Your sparring partner for the ServiceNow Certified System Administrator exam.

## 🎯 What is this?

A free, browser-based exam preparation tool for the **ServiceNow CSA certification**.  
No login. No install. No backend. Just open the file and start training.

Like a real sparring partner, it doesn't go easy on you — it simulates real exam pressure, exposes your weak spots, and makes you explain every wrong answer before moving on.

🔗 **[Live demo →](https://tests95.github.io/csa-quiz)**

---

## ✨ Features

- **300 questions** covering all 15 domains of the official CSA blueprint
- **~15% multiple-choice questions** — select all correct answers, just like the real exam
- **4 training modes:**
  | Mode | Description |
  |------|-------------|
  | 🏆 Mock Exam | 30–120 random questions, countdown timer, 70% pass threshold |
  | 📖 Free Study | Browse all 300 questions with answers and explanations visible |
  | ⚡ Flash Quiz | 30 questions, no timer, instant feedback |
  | 🎯 By Topic | Focus on one specific domain |
- **Full post-exam review** — filter by Correct / Incorrect / Skipped, with detailed explanations
- **Question navigator** — jump to any question, flag for review
- **Timer warnings** — yellow at 5 min, red blinking at 1 min remaining
- **Like counter** — shared across all users
- **Zero dependencies** — single HTML file, works offline

## 🚀 Usage

### Option 1 — Online
Visit the live site: **[tests95.github.io/csa-quiz](https://tests95.github.io/csa-quiz)**

### Option 2 — Local
